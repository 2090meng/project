# 情绪转像素画 Skill

## 触发条件
用户说"生成像素画"、"情绪像素"或输入情绪后请求可视化展示。

## 执行逻辑

### 第 1 步：接收情绪
获取用户输入的情绪文字或已有的情绪分析结果（8 维分数）。

### 第 2 步：映射配色方案
根据主导情绪选择基础色 + 两个辅助色：

| 情绪 | 主色 | 辅色 1 | 辅色 2 |
|------|------|--------|--------|
| 快乐 | #FFD93D | #FF6B6B | #FF9F43 |
| 平静 | #74B9FF | #A8D8EA | #81ECEC |
| 悲伤 | #6C5B7B | #355C7D | #3E3E3E |
| 愤怒 | #E74C3C | #C0392B | #F39C12 |
| 期待 | #F38181 | #FFEAA7 | #DFE6E9 |
| 恐惧 | #6A0572 | #9B59B6 | #BDC3C7 |
| 厌恶 | #78A178 | #27AE60 | #7F8C8D |
| 信任 | #A8E06C | #2ECC71 | #81C784 |

### 第 3 步：生成 16×16 像素网格
- 使用 CSS Grid 布局，每个像素 18px × 18px
- 背景白色，像素间加 1px 间隙
- 情绪分数 0-3 填充 1-3 个像素，4-7 填充 4-6 个，8-10 填充 7-9 个
- 像素位置根据输入文字哈希确定排列（相同输入总是相同画面）

### 第 4 步：输出 HTML
返回 `<style>` + `<div>` 完整 HTML 片段：
- 每个像素用 `transition` 动画（opacity 0→1，stagger 0.03s 递进）
- 像素画下方标注情绪名称 + 主导颜色
- 使用 `.emotion-pixel-grid` 作为容器 class
