# 情绪诗歌生成 Skill

## 触发条件
完成情绪分析后自动触发，或用户说"写首诗"、"情绪诗"。

## 执行逻辑
1. 获取当前情绪类型和强度（8维分数）
2. 优先调用 DeepSeek API 生成个性化诗歌
3. API 不可用时使用预设模板降级
4. 输出 4-6 行短诗

## API 调用配置
- 模型: deepseek-chat
- 温度: 0.9（高创造性）
- 要求: JSON 格式返回，每行 ≤10 字，末尾双行押韵
- System prompt: "你是一位情感细腻的诗人，请根据情绪分析结果创作短诗"

## 降级模板
- **快乐**: "阳光洒满心房/笑容如花绽放/风轻轻吹过/带走所有忧伤"
- **平静**: "心如止水/万物皆宁/云卷云舒/不问归期"
- **悲伤**: "细雨落窗前/心事付流年/夜深人未眠/梦里寻不见"
- **愤怒**: "烈焰燃胸中/怒涛卷长空/待到风平后/心事与谁同"
- **恐惧**: "暗影渐逼近/迷雾笼心间/一束微光亮/照见来时路"
- **惊讶**: "世界翻了个身/色彩满溢而出/睁开眼睛看见/奇迹就在此处"
- **厌恶**: "退后一步的距离/看清了什么值得珍惜/浊水终会流去/清泉自在心底"
- **期待**: "晨曦微露/星光未逝/站在今天和明天之间/满心是你"
- **信任**: "根深扎入大地/枝叶伸向天空/风来过雨来过/我依然在这里"

## 输出格式
返回 JSON:
```json
{
  "lines": ["第一行", "第二行", "第三行", "第四行"],
  "rhyme_scheme": "AABB",
  "dominant_emotion": "快乐"
}
```
