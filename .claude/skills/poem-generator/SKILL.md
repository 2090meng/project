# 情绪诗歌生成 Skill

## 触发条件
完成情绪分析后自动触发，或用户直接说"写首诗"、"情绪诗"、"一首诗"。

## 执行逻辑

### 第 1 步：获取情绪数据
从已有分析结果中提取 8 维情绪分数和主导/次主导情绪。

### 第 2 步：生成诗歌
调用 DeepSeek API（deepseek-chat）生成个性化诗歌：
- 温度 0.9（高创造性）
- System prompt：你是一位情感细腻的诗人，请根据情绪分析结果创作 4 行短诗
- 要求：每行 ≤10 字，第 2/4 行末尾押韵
- 返回 JSON：`{"lines": ["行1", "行2", "行3", "行4"]}`

### 第 3 步：降级处理
API 不可用时使用预设模板：

| 情绪 | 诗歌 |
|------|------|
| 快乐 | 阳光洒满心房 / 笑容如花绽放 / 风轻轻吹过 / 带走所有忧伤 |
| 平静 | 心如止水 / 万物皆宁 / 云卷云舒 / 不问归期 |
| 悲伤 | 细雨落窗前 / 心事付流年 / 夜深人未眠 / 梦里寻不见 |
| 愤怒 | 烈焰燃胸中 / 怒涛卷长空 / 待到风平后 / 心事与谁同 |
| 恐惧 | 暗影渐逼近 / 迷雾笼心间 / 一束微光亮 / 照见来时路 |
| 惊讶 | 世界翻了个身 / 色彩满溢而出 / 睁开眼睛看见 / 奇迹就在此处 |
| 厌恶 | 退后一步的距离 / 看清什么值得珍惜 / 浊水终会流去 / 清泉自在心底 |
| 期待 | 晨曦微露 / 星光未逝 / 站在今天和明天之间 / 满心是你 |
| 信任 | 根深扎入大地 / 枝叶伸向天空 / 风来过雨来过 / 我依然在这里 |

### 第 4 步：格式化输出
返回诗歌文本 + 情绪标注，以优雅的 HTML 格式展示。
