# 情绪转像素画 Skill

## 触发条件
用户说"生成像素画"、"情绪像素"或输入情绪文字后请求视觉化。

## 执行逻辑
1. 接收用户输入的情绪文字
2. 分析情绪关键词，映射到配色方案
3. 生成 16×16 像素网格（CSS Grid 实现）
4. 返回带动画的 HTML/CSS 像素画展示

## 情绪-颜色映射
- **快乐**: #FFD93D, #FF6B6B, #FF9F43
- **平静**: #74B9FF, #A8D8EA, #81ECEC
- **悲伤**: #6C5B7B, #355C7D, #3E3E3E
- **愤怒**: #E74C3C, #C0392B, #F39C12
- **期待**: #F38181, #FFEAA7, #DFE6E9
- **恐惧**: #6A0572, #9B59B6, #BDC3C7
- **厌恶**: #78A178, #27AE60, #7F8C8D
- **信任**: #A8E06C, #2ECC71, #81C784

## 像素规则
- 16×16 网格，每个像素 18px×18px
- 背景 #FFFFFF，像素间有 1px 间隙
- 分数 0-3 填充 1-2 个像素，4-7 填充 3-5 个，8-10 填充 6-8 个
- 像素以随机但确定性的方式分布（种子基于输入文字）

## 输出格式
返回 HTML 代码块，包含：
1. 一个带有 `style` 标签的完整 HTML 片段
2. CSS Grid 布局的 16×16 像素网格
3. 每个像素带有 `transition` 动画（opacity 0→1，staggered delay）
4. 像素画下方标注情绪名称和主导颜色
